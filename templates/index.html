📖 CARA PENGGUNAAN

Setup dalam 4 Langkah

Dari download sampai aktif di Point Blank — cepat dan mudah.

01
📥
DOWNLOAD
Klik tombol Download. Pilih Script V1 atau V2 sesuai lisensi yang kamu punya.
02
🛡️
RUN AS ADMIN
Klik kanan file .exe → Run as Administrator. Jika ada warning antivirus, klik "More Info" → "Run Anyway".
03
🔑
AKTIVASI
Masukkan License Key yang diberikan admin, klik ACTIVATE LICENSE. Timer mulai berjalan saat ini.
04
🎮
MAIN!
Buka Point Blank. Atur delay (rekomendasi 1–10ms), tekan F1 untuk ON/OFF.

⌨️ HOTKEYS

F1 Toggle Macro ON / OFF
1 2 3 Auto ON — Slot Senjata / Pistol / Pisau
4 5 6 Auto OFF — Aman untuk Bom / C-4 / Granat V2
Q / RC Scope Bind — aktif saat ADS / Zoom V2
F6 Load / Save Config — simpan pengaturan V2
F7 Toggle Overlay In-Game (tanpa Alt+Tab) V2
